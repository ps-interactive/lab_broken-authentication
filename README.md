# Notes On Usage

## `src` and `src.broken`
The folders `src.broken` and `src` contain different versions of the same app, with `src` containing the version that conforms to modern authentication practices.
Please review both and compare them to get an idea of what to do and what not to do.

## Getting Started
Run npm install:
`npm i`

## To Run The Server
In terminal, type
`npm run dev`
This will start `src.broken` by default. To start `src`, run `npm run dev-final`.

## Testing the Server
To test the app, in your terminal, type
`npm run debug`

This will open a text interface that can be used to test the server. The server must already be running via `npm run dev`.
